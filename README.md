# HireX – AI Powered Job Portal 🚀

HireX is a modern **AI-powered job portal** built using the **MERN Stack** that connects candidates and recruiters with intelligent hiring tools.  
The platform includes **AI Resume ATS analysis**, an **AI career assistant chatbot**, recruiter dashboard, admin panel, and real-time notifications.  
This project demonstrates **full-stack development, AI integration, and scalable architecture**.

---

## 🌐 Live Demo
🔗 [HireX on vercel](https://hirex-app.vercel.app/)

## 🧠 Key Features

### 👨‍💼 For Candidates
- Search and filter jobs  
- Apply for jobs with resume upload  
- AI-powered resume ATS scoring  
- Receive job status notifications  
- AI career assistant chatbot  
- Track job applications  

### 🏢 For Recruiters
- Recruiter authentication  
- Post new jobs  
- Manage job listings  
- View applicants  
- Accept / Reject candidates  
- AI ATS analysis of resumes  

### 🛠 Admin Panel
- View all applications  
- Monitor platform activity  
- Manage job postings  
- Platform-level control  

---

## 🤖 AI Features

### AI Resume ATS Scoring
HireX automatically analyzes resumes using AI-powered ATS scoring. It provides:
- Match score  
- Matched skills  
- Missing skills  
- Resume improvement suggestions  

**Powered by:** Ollama AI + LLM models  

### AI Career Assistant Chatbot
Candidates can interact with an AI assistant for:
- Career advice  
- Job search help  
- Resume tips  
- Skill recommendations

## 🔐 Authentication
HireX uses **Clerk Authentication** for:
- User login  
- Secure sessions  
- Role-based access  

**Roles supported:**
- Candidate  
- Recruiter  
- Admin  

---

## 📬 Email Notifications
HireX sends email notifications when:
- Recruiter accepts application  
- Recruiter rejects application  
- Job updates  

**Powered by:** Nodemailer + Gmail SMTP
